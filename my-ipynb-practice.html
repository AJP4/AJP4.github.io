Quadratic Equation Solver and Plotter¶

Import useful modules. Math is used for calculating square roots Matplotlib is used for creating the plots Numpy is simply used to create a list of numbers in a range, we could do this without numpy

In [1]:
import math as math
%matplotlib inline
import matplotlib.pyplot as plt
import numpy as np

Create the function that returns the two roots of the quadratic. Notice the use of the Try/Except statement to capture roots of negative numbers. Roots of negative numbers are possible but we would have to introduce imaginary number ('i')

In [3]:
def solveQuadratic(a,b,c):
    x=[]
    try:
        main=math.sqrt(b*b-(4*a*c))
    except:
        print ("********negative root**********")
        x=[0,0]
        return x[:]

    x.append((-b+main)/(2*a))
    x.append((-b-main)/(2*a))
    x.sort()
    return x[:]

This function returns the value of y for a value x when put through the quadratic equation

In [4]:
def f(x,a,b,c):
    return a*x*x+b*x+c

Calculations¶

Now to do some calculations. Change the values of a,b,c (the coefficients of the quadratic)

In [11]:
a=-1
b=3
c=10

The function "solveQuadratic", returns a list of values of x (two for each root).

In [12]:
x=solveQuadratic(a,b,c)

When I plot the curve on the graph I want the graph to include the x values of the roots plus a little bit either side. So I have introduced variables minX and maxX which are 40% bigger than the two roots returned.

Now prepare a range of x and y points for plotting based on the quadratic function¶

In [13]:
minX=x[0]-abs(x[0]*0.4)
maxX=x[1]+abs(x[1]*0.4)

Next we create a range of xValues in increments of 50 between the minX and maxX values and the then calculate the list of yValues associated with each xValue. This allows us to plot these points below

In [14]:
xValues = np.linspace(minX,maxX,50)
yValues = f(xValues,a,b,c)

minY and MaxY are obtained in order to draw a line to represent the Y-axis

In [15]:
minY=yValues.min()
maxY=yValues.max()

Print the Quadratic Curve using the Module MatPlotLib¶

In [16]:
figure1,ax = plt.subplots(figsize=(10,10))
ax.plot(xValues,yValues, label=r"$y=$"+str(a)+r"$x^2 +$"+str(b)+r"$ x + $"+str(c))
ax.plot([minX,maxX],[0,0],color="black", linestyle = ":")
ax.plot([0,0],[minY,maxY],color="black", linestyle = ":")

ax.annotate("1st root = "+str(x[0]),(x[0],0), xycoords ='data', xytext =(x[0],yValues.max()/2), arrowprops = dict( arrowstyle ='->'))
ax.annotate("2nd root = "+str(x[1]),(x[1],0), xycoords ='data', xytext =(x[1],yValues.min()/2), arrowprops = dict( arrowstyle ='->'))

figure1.suptitle("Quadratic Equation Plot Showing Roots", fontsize=18)
ax.set_xlabel(r"$x$",fontsize = 18)
ax.set_ylabel(r"$y$",fontsize = 18)

ax.legend(title="Quadratic Equation")
